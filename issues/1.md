## Objetivo
Ter o esqueleto navegável com permissões e UX consistente.

## User Story A1
**Como usuário, quero autenticar e ver a interface conforme meu papel.**

### Tasks
- [ ] Setup Laravel 12 + Livewire 4 + Flux UI + Tailwind
- [ ] Layout base (AppShell): Topbar, Sidebar, Content
- [ ] Componentes Flux UI padrões: buttons, badges, toasts, modals, tables, tabs
- [ ] Autenticação (Laravel starter / Breeze / Jetstream sem Teams) com Livewire
- [ ] Middleware de papel + guards
- [ ] Seed: 1 Family, 1 Parent, 1 Child, tarefas demo

### Critérios de aceite
- Parent e Child fazem login e veem menus diferentes.
- Flux UI aplicado consistentemente (inputs, modals, tables).
